# Mockup to HTML App

## About
Tool designed to create a web page based on a hand drawn image. For this purpose, I used computer vision methods with the help of functions implemented within the OpenCV library.

## Functional requirements
1. Recognition of several elements such as buttons, images, input fields, text elements.
2. System generates HTML elements based on some predefined rules, e.g. the image is represented by a rectangle with a "X" in the middle.

## Notice
This project is developed as part of my bachelor thesis work at the Faculty of Electrical Engineering in Sarajevo.
